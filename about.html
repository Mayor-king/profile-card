About Me

My Journey, Reflections, and Aspirations

Bio

Hello! I'm Mayowa Lawal, a passionate Frontend Developer and aspiring Software Engineer based in Lagos, Nigeria. My journey into tech began with a fascination for how websites work and has evolved into a deep commitment to creating beautiful, accessible, and user-friendly web experiences.

I believe in the power of clean code and thoughtful design to solve real-world problems. When I'm not coding, you'll find me on the basketball court, reading tech blogs, or spending quality time with friends. I'm driven by continuous learning and the excitement of building things that make a difference.

Goals in This Program

I'm embarking on this journey with clear objectives that will shape my growth as a developer:

  • Master modern JavaScript frameworks and libraries to build scalable applications
  • Develop a strong foundation in data structures and algorithms for technical interviews
  • Build a diverse portfolio of projects that demonstrate real-world problem-solving skills
  • Learn industry best practices for code quality, testing, and deployment
  • Network with fellow developers and mentors to accelerate my learning journey
  • Gain confidence in tackling complex technical challenges independently
  • Contribute to open-source projects and give back to the developer community

Areas of Low Confidence

Growth requires honest self-assessment. Here are areas where I'm working to build more confidence:

Backend Development

While I'm comfortable with frontend technologies, backend development with databases, server-side logic, and APIs still feels intimidating. I'm actively working to bridge this gap through hands-on projects and structured learning.

System Design & Architecture

Understanding how to design scalable systems and make architectural decisions for large applications is an area I need to strengthen. I'm studying system design patterns and learning from experienced engineers.

Technical Interviews

The pressure of live coding interviews and whiteboard sessions makes me nervous. I'm practicing algorithms daily and working on communicating my thought process more clearly.

Advanced JavaScript Concepts

Topics like closures, prototypal inheritance, and asynchronous programming patterns sometimes trip me up. I'm committed to deepening my understanding through deliberate practice and building projects that leverage these concepts.
